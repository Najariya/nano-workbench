# Chrome Web Store Assets

Generated assets for the Chrome Web Store listing.

## Files

- `screenshots/01-page-summary.png` - 1280 x 800 screenshot.
- `screenshots/02-document-review.png` - 1280 x 800 screenshot.
- `promo/small-promo-tile-440x280.png` - required 440 x 280 small promotional tile.
- `promo/gemini-nano-workbench-promo.mp4` - optional local promo video. Chrome Web Store expects a YouTube URL, so upload this video to YouTube and paste the video link if you want to use the video field.
